# User Service TODO

## Суть сервиса

Сервис отвечает за профиль пользователя, публичные пользовательские данные и операции, связанные с балансом.

## Заметки

- Сервис зависит от JWT, который выдает `auth-service`.
- Сервис использует отдельную БД PostgreSQL `bingo_users`.
- Профиль пользователя хранится по `auth_user_id`.
- Сейчас таблицы создаются автоматически при старте сервиса/
